## Objetivo
O objetivo desse estudo é utilizar a análise descritiva em conjunto com o método EDA para a formentação de hipóteses de maneira que as mesmas possam ser testadas em
análises preditivas. Todo este caso de estudo tem por sua vez foco apenas na análise descritiva, logo nenhum modelo preditivo será utilizado para este caso.
